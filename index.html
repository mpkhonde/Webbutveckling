Om mig

Mitt namn är Moise, och jag har en bakgrund som lastbilschaufför. Jag har alltid haft en passion för teknik och problemlösning, vilket ledde mig till beslutet att byta karriär och satsa på att bli Javautvecklare. Jag studerar nu aktivt och bygger projekt för att stärka mina färdigheter inom mjukvaruutveckling.

Teknik jag studerar

Jag specialiserar mig på Java-programmering och har byggt en solid grund inom objektorienterad programmering (OOP). Min inlärning fokuserar på att utveckla moderna lösningar med hjälp av populära verktyg och ramverk som Spring Boot och JDBC.

Nuvarande fokus

  • Spring Boot - Utveckling av backend-system
  • JDBC och SQL - Dynamisk databehandling
  • Maven - Projektberoenden och byggprocesser

Verktyg jag använder

Min utvecklingsmiljö består av IntelliJ IDEA, tillsammans med Git och GitHub för versionskontroll. Jag använder också Docker för att skapa containerbaserade utvecklingsmiljöer.
